Inspección Pre Operativa

Guía de revisión técnica para equipos

Uso de Check-list

Se recomienda llevar una secuencia para completar el check-list, con el fin de llegar a inspeccionar todos los componentes y sistemas del equipo.

Inspección visual

Se recomienda realizar una inspección visual caminando alrededor del equipo, revisando los componentes de la estructura del equipo, neumáticos, luces, etiquetas, controles, entre otros.

Inspección de horquillas y carruaje (bastidor)

Se recomienda realizar la búsqueda de posibles criterios para descartar la operatividad del implemento de trabajo, como desviaciones en las horquillas, rajaduras, fisuras, desniveles, etc.

Inspección de componentes hidráulicos

Se recomienda realizar una inspección a las mangueras hidráulicas, cañerías y conexiones, en búsqueda de posibles fugas, asimismo revisar el sistema de elevación.

Inspección de la cabina

Ingresar al compartimiento del operador y realizar la verificación de los pedales, luces, espejos, dispositivos de seguridad, bocinas, mandos y controles, entre otros.
